## Run

In order to run Shadow Engine on your computer you need these things:

- A 64-bit Windows 10 Machine
- [Node.js](https://nodejs.org "Node.js")
- [Git](https://git-scm.com/)
- G++/GCC/MinGW??
```
    git clone https://github.com/77Z/Shadow-Engine.git
    cd Shadow-Engine
    npm i
    npm run rebuild
    npm start```
